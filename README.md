# Student Result Management System (C)

This is a simple **menu-driven Student Result Management System** developed in C.

## Features
- Add student marks
- Display student results
- Calculate total and percentage
- Show Pass/Fail status

## Concepts Used
- Structures
- Arrays
- Loops
- Functions
- Menu-driven interface
